# Background I want to learn a bit more about myself so I realized, why not “scrape” my own personal data and see what I can learn or at least visualize. My first foray will be into studying my Amazon web purchases. For better or for worse, Amazon has made it incredibly easy to purchase anything you need, but it all began with books. So lets take a look at my Amazon book purchase history.

I always said my computer is my life… so lets find out what it has to say about me.

First, here is the code I used to scrape everything in my Documents directory and create a list of files with all of their info:

setwd("~/Documents")
all_dir <- list.dirs()
file_data <- NULL
for(d in all_dir){
  files <- list.files(path=d)
  for(f in files){
    file_data <- rbind(file_data,as.data.frame(file.info(paste(d,f, sep="/"))))
  }
}
